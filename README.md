# 💰 AI Personal Finance Copilot

## 🚀 Overview
An AI-powered system that analyzes personal finances, detects anomalies, forecasts spending, and provides smart financial advice using Multi-Agent AI and RAG.

---

## 💡 Features
- Expense categorization
- Anomaly detection
- Spending forecast
- RAG-based financial advice
- Streamlit dashboard

---

## ▶️ Run
pip install -r requirements.txt  
streamlit run app.py
